
**Disney Movie App**

A mobile streaming application inspired by Netflix, built to provide Disney movies, series, and animations.

**About**

The Disney Movie App is a semester project for the Mobile Application Development course. It demonstrates modern mobile app development using a clean architecture and popular technologies.

**Features**

Browse Disney movies and shows
Search and filter content
User profiles and watchlist
Continue watching across devices

**Architecture**

MVVM (Model–View–ViewModel)
Model: Data & APIs
View: User Interface
ViewModel: App logic

**Tech Stack**

Frontend: React Native
Backend: Node.js, Express
Database: MongoDB
Deployment: Render
